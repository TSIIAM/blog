Front-End Handbook 2017

Navegando en la red se encuentra de la mano de los Front-End masters, el libro de mano que todo programador front-end debe tener este año 2017.

Comenzando por Front-End Handbook 2017 dispone de no solo cientos de recursos que puedes obtener en la red, sino de una curricula y plan de estudio para este año.

Como los Front-End Developers son hechos:

Como exactamente uno llega a ser un Desarrollador Front-End?

Aca una curricula que puedes seguir como plan de estudio:

  1. Aprender como funciona la Web:DNS,URLs, HTTP,redes,navegadores,servidores,JSON, data APIs, HTML, CSS, DOM y JavaScript.No es necesario estudiarlo a fondo, lo suficiente para tener una idea de como funcionan en conjunto.
  2. Aprender HTML.
  3. Aprender CSS.
  4. Aprender Javascript.
  5. Aprender DOM .
  6. Aprender JSON y APIs de datos.
  7. Aprender los fundamentos del diseño de Interface de Usuario y Experiencia de Usuario(UI/UX).
  8. Aprender CLI, Linea de Comando.
  9. Aprende la practica de Ingenieria de Software(Aplicaciones/diseño/arquitectura,GIT,testing,monitoreo,automatización,calidad de codigo,metodologias de desarrolo).
  10. Aprende un Framework de Javascript(Angular,Webpack,React,Redux).
  11. Aprende Node.js
