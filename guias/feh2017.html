Front End Developer 2017 de Cody Lindley

Recomiendo a todos se tomen un tiempo para leer Front End Handbook 2017 e informarse sobre la mayoria de las técnologias modernas y donde aprender sobre ellas.

Este libro posee varios recursos de Utilidad para los que comienzan y quieren aprender a llegar a ser un desarrollador front end. El objetivo de esta nota es presentar varios recursos utiles para facil acceso, todos los recursos gratuitos que ya puedes comenzar a utilizar pero este no sustituye la lectura del mismo ya que se omiten muchos puntos que son necesarios leer.
