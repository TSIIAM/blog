Aprendiendo PHP y SQL

PHP es el lenguaje de servidor más utilizado hoy en día en toda la web. PHP The right Way, es un libro HTML de buenas practicas del uso de PHP, introducción, practicas,bases de datos, seguridad, testing,etc.

Conjunto a PHP esta SQL, Strutctured Query Language, el cual seria el lenguaje de las bases de datos. SQL essential contiene un buen curso de SQL 101 para introducirse en el lenguaje.Luego tambien puedes leer SQL for nerds que contiene más material de modelado de datos, consultas,etc. Luego un curso más moderno de Khan Academy Intro to SQL:Querying and managin data enseña a utilizar SQL en videos y ejercicios practicos, a su vez contiene Desafios para poner en practica lo aprendido como crear una base de datos de lista de libros, exitos de taquilla, listas TODO y muchos más.

Aprendiendo GIT y Command Line

Git La guia sencilla, te permite en pocos minutos estar trabajando con GIT.

Primero comentar que puedes descargar de manera gratuita el libro Pro Git, GIT es un software de control de versiones que te permite llevar un control de las diferencias que llevas de un codigo a otro, puedes utilizar GitHub para crear repositorios en interent, de manera de respaldar todo tu codigo en un punto o más adelante trabajar en equipo de manera remota.

¿Qué son los Gists? Es un plataforma para snippets, o lo que es lo mismo, pequeños pedazos de código reusables con una funcionalidad concreta. Estos snippets serían ideales tenerlos en un sitio donde se puedan compartir, almacenar y evitar que se pierdan en el olvido, pero donde crear un repositorio git, parece ser excesivo. Puedes leer un Gist tutorial que explica mejor como utilizarlos de manera efectiva.
