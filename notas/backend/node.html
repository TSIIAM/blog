Recursos sobre Node.js

Node.js es un entorno en tiempo de ejecución multiplataforma, de código abierto, para la capa del servidor (pero no limitándose a ello) basado en el lenguaje de programación ECMAScript(javascript), asíncrono, con I/O de datos en una arquitectura orientada a eventos y basado en el motor V8 de Google.

Puedes comenzar con una introducción a Server Side Introduction en MDN. Interiorizarte con los distintos backend Web Frameworks .

Como comenzar con Node?

Para comenzar puedes descargarlo en Node.js.Puedes encontrar la documentación en Node api

Otro buen sitio para aprender Node y otras materias es en Node School.

npm install -g learnyounode

Tambien puedes aprender node mediante la documetación de node en stack overflow.

Tutoriales de node nodebeginner y nodeguide que puedes leer en linea.
