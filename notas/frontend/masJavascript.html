Javascript Para todo los gustos.

Para los que estan aprendiendo Javascript traigo varios recursos utiles, comenzando con JS for cats que es un libro web que enseña lo basico de JS de manera de explicarselo hasta a el gato. Si quieres comprender mejor de que se trata javascript puedes en JS the right way donde se explica a nuevos programadores y a los más experimentados de que se trata javascript, sus patrones, buenas practicas, frameworks, utilidades para testing, noticias, lecturas, etc. Luego un buen libro gratuito es Speaking Js.

Practicar con desafios

Una vez te sientas listo para un desafio, existen multiples sitios con ejercicios para probar lo aprendido y aprender ejercitando el cerebro por ejemplo en Code Wars donde puedes entrenar mediante Katas, ejercicios creados por la comunidad de programadores para ejercitar y practicar un lenguaje deseado. Luego otro sitio muy recomendado es CoderByte que fue creado para preparar a los programadores para lidear con entrevistas técnicas, en el sitio puedes encontrar mini cursos para javascript, ruby y python con tests y videos donde se explican diversos temas.

Video Cursos

Para los que tienen menos tiempo cerca del teclado y quiere seguir algun curso por video, les recomiendo un par de buenos cursos gratuitos, aunque en ingles, pueden ver a programadores explicandote el codigo que van programando en learn to program TV donde les recomiendo cursos como aprender javascript, luego Javascript Avanzado, la libreria jQuery y como programar de manera asincrona con Ajax.
