Javascript

Javascript es el lenguaje de programación de la web de lado de cliente que nos permite modificar el comportamiento de las paginas. Este lenguaje de programacíon debe ser el arma preferida de cualquier desarrollador front end, speakingjs se puede leer gratis en HTML, tambien es facil obtenerlo en pdf, es un libro muy recomendado para aprender Javascript.

You Dont know Javascript A book series on JavaScript, de Kyle Simpsons, contiene 6 libros gratuitos, de Introducción a Javascript que no solo explica el porque sino tambien el como, el primer libro te introduce a la sintaxis basica de javascript, los siguientes libros tratan de temas más intermedios que todo programador de javasript deberia saber. YDKJS se puede leer gratis en github y es una colección muy recomendada para principiantes ya que pose temas concretos, tambien el humor del autor ayuda a entender como funciona.

Javascript Libros

Traigo un par más de libros de javascript para obtener más conocimiento, DOM Enlightenment es un excelente libro de Cody Lindley enfocado al modelado del objeto documento.

Aprendiendo Design Patterns de Addy Osmanim que puedes leer en version html, enfocado a patrones de diseño.

Tambien Open Data Structures de PAT MORIN enfocado a estructuras de datos. Estos tres libros son algo más intermedios, no estan enfocados a quienes estan comenzando, pero es bueno ya tenerlos a mano, ya que seran una gran fuente de conocimiento para mejorar su desarrollo como programador front end.

Practicar Online

Hoy en dia hay cientos de formas de practicar online por ejemplo las más reconocidas paginas jsFiddle y Codepen te permiten crear de manera online tus diseños html/css/javascript, actualizar cambios, utilizar librerias y compartir tus diseños de manera gratuita.
