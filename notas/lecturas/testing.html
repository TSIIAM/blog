Recursos para aprender sobre testing.

Hoy para compartir traigo varios recursos para aquellos interesados a aprender sobre testing, comenzando por Guru99 Learn Software Testing es un sitio con varios tutoriales aprender sobre testing en general, desde herramientas, distintos tipos de testing y si tu objetivo es aprender sobre testing es el sitio mas recomendado .

Luego una guia de Tutorialspoint sobre Software Testing da una introduccion al Software testing que junto al Software Testing Dictionary te sera de gran utilidad para comenzar a aprender testing.

Si te interesa seguir una revista sobre Testing Testing Magazine te ofrece de manera gratuita una revista online sobre testing.

Para completar si te interesa recibir un titulo estudiando de manera online Aprende.org te ofrece un curso de Tester en español, cabe mencionar que Aprende.org ofrece cursos en español para varias carreras más.

Finalizando, si quieres adquirir algunas apps para android sobre testing, busca Guru99 y ISQTB test mentor y Challenger.

Esto eso todo por esta semana, voy a tratar de recopilar varias guias y recursos para proximas entregas para facilitar la busqueda.
