Donde practicar con ejercicios

Parte de aprender a programar es practicar, realizar ejercicios para aplicar o aprender patrones y algoritmos. Leer, buscar y preguntar para lograr resolver problemas.Por eso hoy comparto un listado de paginas donde pueden ejercitar la mente.

Comenzando con HackerRank donde te encuentras con varios desafios, hay varios que son ejemplos de entrevistas de trabajo, tambien cursos y distintos temas.

Luego un sitio que recomiendo ya que lo utilizo regularmente, Code Wars te ofrece katas, asi como quienes practican artes marciales mediante ejercicios repetitivos, los katas son problemas que debemos resolver con el lenguaje que desees, a medida que resuelves ejercicios vas subiendo rangos como en el karate.

Si quieres simplemente entretenerte un rato mientras practicas, una manera para aplicar programación jugando es en Code Combat es una aventura en la cual vas realizando ejercicios, a medida que avanzas tu personaje aprende técnicas nuevas(patrones) que puedes reutilizar y combinar para resolver niveles más complejos. Otro "juego" que existe es Codingame, este ya no es tan juego para principiantes, sera necesario tener una base para poder comprender como resolver los puzzles que te presentan.

Puedes acceder con tu cuenta de gmail en HackerEarth y puedes acceder a ejercicios de distintos lenguajes, con una interface sencilla y instrucciones que te explican que debes realizar.

Si quires incluir un poco de matematica en tu practica Project Euler te sera util, con problemas de distintas dificultades, puede ser un buen recurso para aquellos que desean realizar un ejercicio con más libertad, sin interfaces graficas ni accesorios, solo un problema por ejemplo encontrar los multiplos de 3 y de 5 debajo de 1000.

Si te interesa un listado de ejercicios con más instrucciones, ejercicios de Code chef te pueden ayudar para ejercitar y si no encuentras la solución la pagina posee las soluciones de otros programadores.

Otro sitio es LeetCode donde puedes registrarte para realizar ejercicios, competir y aprender.

Recomiendo CoderByte, este sitio fue creado con la idea de asistir a los programadores para prepararse para entrevistas, posee cursos y tutoriales.Posee un curso de fullstack developer gratis.

Por ultimo recomiendo Code Fights donde pueden realizar practicas, desafios, torneos y un mapa para ir aprendiendo a programar.
