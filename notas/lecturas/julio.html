Recopilando los ultimos meses.

Llego Julio y los meses pasan tan rapido, queria tomarme un tiempo para resumir estos ultimos 3 meses y comparar mis avances desde que comenze en enero.
Este ultimo mes puse en marcha en llevar conmigo todos los dias un registro de lo que voy estudiando, leyendo o investigando de manera de ir un poco más ordenado y poder centrarme en completar libros, tutoriales y recursos que me encuentro.

Recuerdo mes a mes hasta Mayo colocaba un pequeño resumen en mi perfil de mi blog para llevar cuenta de lo aprendido, en Junio por cuestiones de tiempo y desorden, no pude completar tantos objetivos como hubiera querido.

Entonces de Junio a Julio realice uno de los cambios en mi manera de almacenar recursos, pase todo mi hoja de calculos google drive a Boostnotes, en boostnotes puedo almacenar notas y snippets de codigos, las notas puedo almacenarlas en markdown de manera de mejor organizar todo. Sigo en proceso de actualizar todas las notas, mejorar como estan organizadas y compartir mis recursos en un Repositor de Github para que todos puedan usufructuarlo.

Como meta de Julio, completar minimo una rutina de dos pomodoros de programación, un pomodoro de ingles y un pomodoro de testing por dia y un libro por mes. Este mes tratar de Completar los libros de You Dont Know Javascript, que me habia detenido en This & Object, la ultima vez me habia resultado sumamente complicado entender varios temas clases y objetos en js y lo habia postergado en Marzo.Luego de bastante más experiencia, pude comprender mejor la programación orientada a objeto con javasript y completar el libro aprendiendo más sobre OOP y prototipos.

Otro libro que me recomendaron en Free Code Camp, Javascript Allonge, me aplique a tratar de leerlo, aunque pude entender algunos temas, el resto del libro no pude comprenderlo por completo, me dio una mayor comprensión sobre la programación funcional, pero estoy todavia muy verde para entenderlo todo, por lo que lo postergue para futuro. Trato de sumergirme en varios temas que abarcan el desarrollo profesional y un rol que todavia no lograba comprender era el Devops, luego de leer State of DevOps 2017 en la pagina de puppet, pude comprender un poco mejor de que responsabilidades tiene, la similitud con los sysadmins y algunas herramientas actuales que usan. Informandome sobre criptodivisas mediante un pdf de Bitcoin de btcfacil.com y Ethereum for beginners.

Resumiendo Julio

Aplicaciones que utilizo para practicar día a día como por ejemplo, soloLearn, codemurai, enki y encode, complete todo encode, gracias a la aplicación puse en practica y comprendi mejor como usar .map, .reduce y .filter, tambien clases y objetos y el uso de this, .apply y .call.En codemurai pase por los fundamentos de JS, a ES6, typescript, MongoDB, Node y comenze con Java y React.

Para leer en mi tiempo y no perder lecturas, en mi telefono comparto las paginas en modo imprimir, logro compartirme los pdf y seguir leyendo en cualquier lugar.Ejemplos como el articulo de Working with JSON data en Mozilla Developer Network. Otro libro que me encontre, era el tutorial de javascript de tutorialspoint, luego de completar 388 paginas, no lo recomiendo, no puedo saber si todo lo teorico estaba correcto, pero la mayoria de los codigos de ejemplos daban error o no funcionaban como querian demostrar, pude tomar algunos elementos sobre DOM, pero no vale la pena.

Este mes de julio comenze a sentarme y estudiar node.js, lo bueno que tiene es la mayoria de recursos practicos que hay como node school, es muy distinto a php en backend y me falta muchas horas de estudio, pero fue bueno para comenzar a poner en practica npm.

Todo esto sin perder de vista el Obligatorio de Programación 1 que tenia que entregar a principios de julio, con bastante empeño pude sentarme y terminar el obligatorio aprendiendo muchisimo sobre estructuras de datos, objetos, constructores, funciones, programar todo con javascript.Del obligatorio trabaje en la interface de un cliente para que pueda crear consultas medicas, ver las consultas generadas y modificar su perfil, hay mucha razon en decir que se aprende mucho más programación trabajando en proyectos practicos y tratando de resolver problemas reales.

Otro beneficio de compartir los pdf para luego, fue completar varias publicaciones de Medium que me resultaron interesantes.Ejemplos como How to experience powerfull education free of charge(7ideas), A beginner Javascript Study Plan que aconseja FreeCodecamp para aprender HTML/CSS/JS Bootstrap y jQuery, complementar con CodeAcademy para reforzar temas, seguir con Understanding the Weird Parts de Tony Alicea en Udemy y You Dont Know Javascript de Kyle Sympson leer Scope & Closures y This & Object.

Luego organize mejor mi bibioteca de libros en una libreria compartida de google drive, puse en marcha y baje varios pdf de la documentación de javascript en stackoverflow, muy recomendado los temas que hay para estudiar, como por ejemplo Asynchronous Javascript and XML AJAX de stackoverflow(aunque hoy en día se utiliza mucho más JSON).

Luego de terminar el obligatorio pude tener más tiempo para otras lecturas varias, ejemplo de medium que puedo compartir como Answering the questions every new developer Asks, FreeCodecamp related questions, TOP 7 javascript frameworks, Why I choose React over VUE?, Building a Javascript Development Enviroment ? y A Study plan to cure Javascript Fatigue. Otros como How to grow Junior Developer, 5 steps mini guide to launch a profitable startup y What is an API? in english please. De stackoverflow la documentacion sobre la consola del navegador, Console, de codeBurst, A brief History of Javascript from ES3 a ES6. De Mozilla Developers Network una introducción a Node y Express, Medium. Javascript a first Class language at last, FCC. How to go from hobbyist to professional Developer.

Terminando Junio habia una oferta de Code School de un año pago a 50% que tenia mis dudas sobre si valia la pena, ya habia probado cursos gratuitos como Try jQuery y Try Git que fueron buena introducción, luego de probar Try Ruby, me decidi por pagar el año y continuar todo la formación de HTML/CSS y Javascript. Puedo asegurar Code School vale cada dolar, los videos son entretenidos, claros y consisos, los desafios funcionan correctamente no como los buggeados de code academy, si estas comenzando a aprender a programar, code school es la mejor herramienta, no solo tiene varios cursos gratis para probar, tiene varios intermedios y más avanzados, explicando cada tema con desafios para poner en practica lo aprendido y varios lenguajes para aprender.Lo unico que le falta para ser la mejor opción es tener una app android( tiene una iOS) y que tenga más proyectos practicos como tiene Code Academy.
