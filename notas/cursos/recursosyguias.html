Recursos y guias gratuitos para nuevos Programadores.

Todos buscamos mejorar y sin duda para mejorar hay que practicar mucho, entonces aqui les traigo varios recursos, guias y cursos que te pueden ayudar a volverte un ninja developer. En lo posible todo aqui estara gratis.

Recomiendo ya comienzes a aprender GIT, git es un sistema de versionado, gracias a Github puedes registrarte y almacenar todo tu codigo en repositores, por el momento puedes aprender lo basico como alojar tu propia pagina, te sera de gran utilidad en tu carrera como profesional. Puedes comenzar con TryGit en Code School, este es un curso gratuito, de manera interactiva aprender las bases practicas de git.

Para mejorar los fundamentos de Git tambien tienes Simple Git un sitio web donde se te explica los comandos más usados de git y si quieres puedes ver los videos de Git Foundations.

Mientras aprendes git ya deberias estar aprendiendo las bases del desarrollo web, pero donde comenzar?. El mejor recurso que tienes es mediante Mozilla Developers Network(MDN) donde se te explica, HTML, CSS, Javascript de nivel basico a intermedio, tambien más adelante es buen recurso para materiales más avanzados como seguridad, desarrollo en servidores,etc. Otros recursos que te pueden servir para practicar son W3 Schools y W3 Resources.

Si quieres aprender con más practica puedes usar Code Academy para aprender HTML y CSS, tambien como crear tu propia pagina y publicarla. Si te interesa tener más practica puedes usar Aprender HTML y CSS con Shay Howe y aprender mediante pequeños projectos practicos en projectos de General Asembly DASH de manera gratuita.

Si te recomiendan algun libro y lo tratas de buscar, puedes buscarlos en varias librerias gratuitas, ejemplo Github free books Library , libros gratis en DevFreeBooks , 20 libros gratuitos en HONGKIAT , Free Tech Books(FTB), en español Libros Web y una cantidad de diversos libros en Programming Motherfucker.

¿El mejor lugar para comenzar gratis? ya lo recomende antes pero lo vuelvo hacer, ya con una cuenta de github pueden registrarse en una variedad de sitios, es como su nueva identidad de programador. Registrense en Free Code Camp donde aprenderan mediante practica y desafios diversas tecnologias como HTML/CSS/Bootstrap, Javascript/jQuery, React, D3.js, Node/npm/Express/MongoDB,etc. No solo es un buen recurso para aprender sino que finalizado pueden recibir certificados y tener un buen portolio de projectos practicos de manera gratuita.Te interesa conocer de otras guias? también puedes seguir Become a FS web developerun repositor de github con muchos más recursos para seguir aprendiendo.

Programador Junior en 5 meses.

Hoy traigo un plan de como aprender a programar en 5 meses de manera estructurada. En el primer mes se tratara comenzando con una introducción a la computación y las redes. Comenzando por Como funciona la internet. y redes de computadoras en youtube.

Luego muy recomendado una introducción a la programación en el curso de EDX, CS50 en youtube o la pagina misma de EDX, este curso tardara más de un mes pero es una excelente introduccion a las ciencias de la computación.

En el segundo mes trata de comenzar con el desarrollo web en Desarrollo web para principiantes y como publicar tu sitio en linea.Debes Empezar con el curso de code school de GIT y el curso de javascript. De esta manera ya comenzarias a estar creando tus primeros sitios y con una buena base de la computación, redes y como mantener tu codigo.

Tercer mes en adelante.

A partir del tercer mes en adelante continuas en Code school con Chrome Dev Tools y los dos pilares de javascript en medium uno y dos donde se trata de el estado actual de javascript, buenas practicas, temas más avanzados y una introducción a crear tus apps.

El cuarto mes comenzar con React un curso de fundamentos en egghead y un tutorial de youtube sobre React

Para terminar el ultimo mes tu primera app en React y comenzar a aprender Redux. Hay que tener en cuenta que Code School y egghead son cursos pagos, hay alternativas gratuitas, pero siendo de pago aun son muy recomendados, también felicitarte ya que conmpletando este plan dispones de un gran listado de técnologias front-end que son muy requeridas en el campo laboral actual.
