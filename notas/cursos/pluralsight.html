Comienzos en la programación Web

Para un buen comienzo en la programación Web de parte de Pluralsight 3 videos front end developing, Front End Web Development: Get Started da una introducción a los lenguajes, comunicación con el servidor, navegadores y performance.

Luego puedes ver Front End Web Development: Career Kickstar da una introducción a la profesion, la industria, como obtener un trabajo y seguir desarrollandote.

Tercero pero no menos importante Front End Web Development Quick Start With HTML5, CSS, and JavaScript detalla más como se crea un HTML, se estila con CSS, se da comportamiento con Javascript y para finalizar como utilizar jQuery.

Pluralsight provee videos de entrenamiento para IT, desarrollo, marketing, diseño, seguridad y otros, es entrenamiento online pago, pero esta la opcion de una prueba gratis. Yo actualmente estoy utilizando una cuenta de 3 meses gratis gracias a los beneficios de Dev Essentials.

No pierdan la oportunidad, para seguir estos cursos de buena calidad,con una buena interfaz que pueden seguir en su celular y descargar los videos en su casa.
